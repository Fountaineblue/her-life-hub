I am building a GitHub Pages website called Her Life Hub for my girlfriend.

It uses:
- index.html
- styles.css
- app.js
- localStorage for saving data
- GitHub Pages for hosting

Current features:
- Home tab with Today card
- Monthly calendar
- To-do list
- Daily affirmation that changes every day
- Useful Links tab
- Budget tab
- Wishlist tab
- Photo Gallery tab with preloaded photos from a /photos folder
- What to Eat tab with a random food generator

The site is hosted from GitHub Pages. I want to keep improving it.
